# Catalog Service

Сервис управления заказами, товарами и клиентами. Реализован на FastAPI + PostgreSQL, запускается в Docker.

## Быстрый старт

### Клонирование репозитория

```bash
git clone <url-репозитория>
cd catalog
```

### Запуск в Docker

```bash
docker compose up --build
```

После сборки образов и запуска контейнеров сервис будет доступен по адресу:

API: http://localhost:8000

Openapi документация: http://localhost:8000/docs

### Остановка сервиса

```bash
docker compose down -v
```

### Примечание

При первом запуске автоматически создаются тестовые клиент и категория, если таблицы пусты.
